# Portfolio Behavior

Purpose:

Present work as evidence rather than as a resume.

Rules:

- Show outcomes before descriptions
- Show process before conclusions
- Use real examples
- Explain decisions
- Show measurable impact
- Show lessons learned

Avoid:

- vague accomplishments
- generic wording
- inflated claims
- unnecessary buzzwords

Ask:

What was the problem?

What was done?

Why was it done?

What happened?

What was learned?
